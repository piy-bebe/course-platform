Создание кликера на JS

назад

Тема

Самостоятельная работа. Разработка интерактивного кликера

Дата

24 декабря

Задание

1. Создать html страничку, подключить css и js к сайту.

2. Сверстать кликер, будет очень хорошо, если для кликера сделаете дизайн в фигме. Пример кликера есть на записи прошлого занятия по веб-разработке. Обязательно создать тег с выводом количества нажатий на кнопку и саму кнопку при нажатии которой будет увеличиваться число. Дополнительно реализуйте вторую кнопку, которая наоборот будет уменьшать число.

3. После верстки и стилизации, когда внешний вид будет готов, приступаем к программированию на JS. Нужно реализовать функционал счетчика, где при нажатии на кнопки будет либо прибавляться число на единичку, либо уменьшаться.

4. Сделать дополнительную кнопку, которая будет обнулять счетчик. Допустим счетчик равен 241, если нажмем на кнопку "обнулить", счетчик будет равен 0.

В этом задании отрабатываются все пройденные навыки, вы соедините 3 технологии: html, css, js. Создадите скелет приложения, добавите стили и оживите сайт с помощью JS. Желаю успехов :)
