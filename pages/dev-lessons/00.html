Задание 1

Создать веб-визитку о себе как о студенте веб-разработки.

Создай шапку, где будет информация о твоем имени с аватаркой. Ниже создай навигацию, используя тег nav -> ul -> li.

В главном теге main, создай секции about (расскажи о себе), skills (напиши текущие навыки, допустим -> HTML и CSS), goals (Цели на 3 - 6 месяцев).

В футере напиши какие-нибудь соц-сети, допусти github аккаунт.

HTML:

  • Корректная структура сайта
  • Заголовки разных уровней (h1-h3)
  • Маркированный и нумерованный списки (ul - ol)
  • Изображение с атрибутами alt, width, height
  • Ссылки с атрибутом href

CSS:

  • Подключение внешнего CSS-файла
  • Использование селекторов: тег, класс, ID
  • Применить минимум 5 свойств из изученных
  • Работа с отступами (margin/padding)
  • Изменение цветов и шрифтов
